## 🔧 インストールとセットアップ

### **ローカル開発**
シルエットジェネレーターをローカルで実行するには：

```bash

git clone https://github.com/yourusername/silhouettegenerator.git

# シルエットジェネレーターディレクトリに移動
cd silhouettegenerator

# 依存関係をインストール

# APIキーを設定（.envファイルを作成）

# シルエットジェネレーター開発サーバーを起動
npm run dev
```
